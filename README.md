# OptiPixel

이미지 해상도(가로/세로)를 유지하면서 파일 용량만 최적화하는 무료 온라인 이미지 압축 도구입니다.

## 기능

- JPG, PNG, WebP 이미지 지원
- 드래그 앤 드롭 / 클릭 업로드
- 압축 품질 슬라이더 (10~99%)
- 원본 vs 압축 결과 미리보기
- 용량 절감 % 시각화
- 클라이언트 사이드 처리 (서버 업로드 없음)

## 시작하기

```bash
npm install
npm start
```

## 빌드

```bash
npm run build
```

## 기술 스택

- React 19
- [browser-image-compression](https://github.com/Donaldcwl/browser-image-compression)
